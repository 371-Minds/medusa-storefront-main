# Back-Alley Stacks

A retro-themed webpage with a hidden "speakeasy" area.

## How to access the speakeasy

To unlock the speakeasy, you need to enter the Konami Code:

`Up, Up, Down, Down, Left, Right, Left, Right, B, A`

## How to run locally

This is a static website. You can run it by opening the `index.html` file in your web browser.

## Technologies used

*   HTML
*   CSS
*   JavaScript
